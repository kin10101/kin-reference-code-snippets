# n8n Setup Guide

## 1. Install Node.js
Download and install Node.js from [nodejs.org](https://nodejs.org).

## 2. Install n8n
Run the following command in your terminal:

```bash
npm install n8n -g
```

## 3. Start n8n
Run:

```bash
n8n start
```

## 4. Open n8n in Your Browser
After startup, open the local URL shown in the terminal.
This is usually:

`http://localhost:5678`
